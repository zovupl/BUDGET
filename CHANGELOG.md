# CHANGELOG

## 2026-07-12 — версия 1.2

Приложение на экране «Домой» само подхватывает новые версии.

- Раньше iPhone держал сохранённую копию и новую версию не искал — обновления
  были видны только при открытии сайта по ссылке в Safari.
- Теперь при каждом открытии (и при возврате из фона) приложение проверяет обновление
  и, если оно есть, само перезагружается на свежую версию.
- Файлы версии: `versions/main_2026-07-12_v1.2.jsx`, `versions/sw_2026-07-12_v1.2.js`.

## 2026-07-12 — версия 1.1

Ручной ввод операций (не только через чат).

- На главном экране под карточкой баланса — кнопки **📤 Трата** и **📥 Доход**
  (видны на всех вкладках).
- Открывается окно ввода: крупное поле суммы, выбор категории плитками с эмодзи,
  заметка, дата (можно внести вчерашнюю трату) и переключатель «кто вносит».
- Подсказки на пустых экранах теперь ведут на эти кнопки, а не только в чат.
- Файлы версии: `versions/app_2026-07-12_v1.1.jsx`.

## 2026-07-12 — версия 1.0

Первая рабочая версия приложения.

- Прототип `family-budget-v2.jsx` перенесён на Vite + React + Tailwind (папка `app/`).
- Хранилище `window.storage` заменено на **Firebase Realtime Database** (узел `/budget`):
  данные общие для всех устройств, обновляются в реальном времени (`onValue`).
- Записи операций и платежей по долгу пишутся каждая по своему пути — записи с телефона
  Мити и телефона Марины не затирают друг друга.
- Убраны вызовы Anthropic API: разбор фраз («500 кроссовки») делает локальный парсер,
  ответы на вопросы («сколько ушло на еду?») считаются на устройстве, интернет не нужен.
- Чат дополнительно понимает платежи по долгу: «отдал 200 долга» → −200 €.
- Общую сумму долга (3000 €) теперь можно менять прямо в приложении, во вкладке «Ещё».
- Показывается плашка «нет связи», выбранный профиль запоминается на устройстве.
- PWA: manifest, service worker (офлайн-кэш), иконки 💰 на градиенте, полноэкранный режим
  на iPhone при добавлении на экран «Домой».
- Автодеплой на GitHub Pages через GitHub Actions при пуше в `main`.
